    网页设计基本技术

    可以把一个网页想象成一栋房子

    HTML - 房子的结构 / 骨架

    HyperText Markup Language，超文本标记语言，一种用于创建网页的标准标记语言（严格意义上来说不算编程语言）。可以使用 .html文件来建立自己的 WEB 站点，HTML 运行在浏览器上，由浏览器来解析。

    CSS - 房子的装修 / 外观

    Cascading Style Sheets，层叠样式表。是一种用来为结构化文档（如 HTML 或 XML 应用）添加样式（字体、间距和颜色等）的计算机语言，CSS 文件扩展名为 .css

    JavaScript - 房子的功能 / 交互

    实现网页的动态效果和交互功能。本学期暂时不涉及

    基础知识

    IP 地址

    上网必须有 IP，没有就连不了网。

    访问网站 = 你的 IP → 找到网站的 IP → 互相传数据。路由器、WiFi、服务器、网站、APP 全都靠 IP 通信。

    内网 IP

    家里 / 学校 / 公司里的地址。只在小范围内有效。比如教室1102 的房间号，出了学校就不认了。

    公网 IP

    全世界唯一的地址。就像你家完整的邮政地址，全国唯一。

    域名

    域名 = 网络世界的 “外号 / 名字”。由于IP地址是数字编码的，不易记忆，所以我们平时上网使用的大多是如www.baidu.com的地址，即域名。www表示万维网（World Wide Web）

    你可以理解成：
    IP = 身份证号
    域名 = 你的名字

    URL

    你在浏览器地址栏输入的那一串东西，就是 URL。

    统一资源定位符（Uniform Resource Locator，URL）其实就是Web地址，俗称“网址”。万维网上的所有文件都有唯一的URL，只要知道资源的URL，就能够对其进行访问。

    如https://www.baidu.com/s/1.html

    • Https://

    协议 → 用什么方式通信（安全的网页）

    • www.baidu.com

    域名 → 你要去哪个服务器

    • /s/

    路径 → 去服务器里的哪个文件夹

    • 1.html

    文件名 → 找哪个网页文件

    总结

    IP = 小区经纬度地址
    域名 = 小区名字
    URL = 详细到门牌号的地址：XX 小区 X 栋 X 单元 X 室

    HTTP与HTTPS

    HTTP

    超文本传输协议（Hyper Text Transfer Protocol，HTTP）

    浏览器和服务器之间的 “说话规则”。互联网上应用最为广泛的一种网络协议。所有的万维网文件都必须遵守这个协议。

    HTTPS

    超文本传输安全协议（Hyper Text Transfer Protocol Secure，HTTPS）

    HTTP 协议穿上了一层 Secure 加密防弹衣。
