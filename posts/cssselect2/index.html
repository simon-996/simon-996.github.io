html-css-5-css盒子模型

发表于2026-04-20|更新于2026-04-20|课件
|浏览量:

🎯 学习目标

  1. 掌握块级元素与行内元素的核心定义与区别。
  2. 熟悉常见的元素分类，能快速识别元素类型。
  3. 理解display 属性如何控制元素显示模式，实现灵活转换。
  4. 结合 div 与 span，深化对页面结构与语义化的理解。
  5. 彻底理解 CSS 盒子模型，能清晰区分 content / padding / border / margin。
  6. 熟练使用 margin 和 padding 控制元素间距与内部留白。
  7. 能独立完成简单布局与文字排版。

什么是 HTML 元素？

HTML 标签包裹的内容统称为元素，是构成网页的基本单元。

根据显示方式和布局特性，元素主要分为两类：

  • 块级元素 (Block-level)
  • 行内元素 (Inline-level)。

块级元素

在页面中独占一行，自动换行，默认宽度撑满父元素。

特性：

  • 独占一行：相邻块级元素上下排列，中间自动换行。
  • 宽高可控：支持设置 width、height、padding、margin。
  • 可嵌套：可以包含其他块级元素和行内元素。
  • 常见场景：页面布局、大区域划分、段落结构。

行内元素

在同一行内依次排列，不独占一行，宽度由内容决定。

特性：

  • 同行显示：相邻行内元素左右排列，空间不足时换行。
  • 宽高不可控：默认不支持设置 width、height，由内容决定。
  • 仅容内容：只能包含文本或其他行内元素，不可嵌套块级元素。
  • 常见场景：文字修饰、小图标、链接、短文本组合。

特殊元素：行内块元素

兼具行内和块级特性，同行显示且支持宽高。

特征：

  • 同行排列（像行内元素）
  • 可设置宽高、边距（像块级元素）

display 属性：控制元素显示模式

HTML 元素的显示模式并非固定，可通过 CSS 的 display 属性灵活转换。

属性值 显示模式 特点 应用场景
block 块级 独占一行、可设宽高 页面布局、大区域
inline 行内 同行显示、不可设宽高 文字修饰、小元素
inline-block 行内块 同行显示、可设宽高 导航菜单、图标按钮
none 隐藏 不显示、不占空间 动态隐藏元素

两个无语义标签

<div> 和 <span> 是 HTML 中最常用、最纯粹的两个无语义容器标签。

无语义：本身不代表标题、段落、图片等含义
纯容器：专门用来包裹内容，方便划分结构 + 设置 CSS 样式

div：块容器

  • =快递盒 / 大箱子（块级）
  • 自己独占一行
  • 别人不能和它并排
  • 可以装文字、图片、其他小盒子
  • 用来划分页面大区域

span：行内容器

  • = 便利贴 / 小标签（行内）
  • 不独占空间
  • 和其他内容挤在同一行
  • 只包裹一小段文字 / 小内容
  • 用来局部修改样式

CSS盒子模型

什么是盒子模型？
所有 HTML 元素都是一个 “方盒子”由 4 部分组成：

  • content 内容区：文字、图片所在区域
  • padding 内边距：盒子内壁到内容的距离
  • border 边框：盒子的边
  • margin 外边距：盒子与其他盒子的距离

padding 内边距（内壁留白）

控制内容与边框之间的距离，让文字不贴着边。

四个属性对应四个方向

  • padding-top 上内边距
  • padding-right 右内边距
  • padding-bottom 下内边距
  • padding-left 左内边距

简写写法（顺时针：上 右 下 左）

特点

  • 会撑大盒子
  • 块级 + 行内元素都能用
  • 背景色会覆盖 padding 区域

margin 外边距（盒子间距）

控制盒子与盒子之间的距离。

四个属性对应四个方向

  • margin-top 上外边距
  • margin-right 右外边距
  • margin-bottom 下外边距
  • margin-left 左外边距

简写写法（和 padding 一样 上右下左）

特点

  • 不会撑大盒子
  • 背景色不会覆盖 margin 区域

border边框

在 CSS 盒子模型中，border（边框） 是位于 padding（内边距）和 margin（外边距）之间的装饰性线条。它就像一个相框，围绕在元素内容和内边距的外围，用于定义元素的边界。

border 的基本语法

简写属性

border: border-width border-style border-color;

border: 边框宽度 边框样式 边框颜色
