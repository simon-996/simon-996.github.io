html-css-4-css选择器

发表于2026-04-03|更新于2026-04-03|课件
|浏览量:

🎯 学习目标

  1. 理解 CSS 的作用、核心价值及其与 HTML 的关系。
  2. 掌握 CSS 的基本语法结构（选择器 + 声明块）。
  3. 熟悉常用 CSS 选择器（标签、类、ID、通配符）。
  4. 掌握 CSS 的三种引入方式（行内、内部、外部）及优先级。

CSS 简介

什么是 CSS？

CSS 全称是 Cascading Style Sheets 层叠样式表。

  • 比喻：
    • 如果 HTML 是网页的“骨架”和“肉体”（毛坯房）；
    • 那么 CSS 就是网页的“皮肤”和“妆容”（精装修）。
  • 分工：
    • HTML：负责结构（标题、段落、图片）。
    • CSS：负责表现（颜色、字体、布局、动画）。

为什么要使用 CSS？

  • 结构与表现分离：HTML 专注于语义化，CSS 专注于美化，便于维护和改版。
  • 提高开发效率：一套 CSS 代码可以控制多个页面的样式。
  • 提升用户体验：实现更丰富的视觉效果和响应式布局。

CSS 基础语法

CSS 规则由选择器和声明块组成。

语法结构

  • 选择器：指定要改变样式的 HTML 元素（如 h1）。
  • 属性：想要改变的样式特征（如 color）。
  • 值：属性的具体设定（如 red）。
  • 声明块：花括号 {} 包裹的一组声明。

注意：

  • 属性与值之间用冒号 : 分隔。
  • 每条声明之间用分号 ; 分隔（最后一行建议也加上）。

CSS 常用选择器

选择器用于“选中”页面中的元素，以便应用样式。这是 CSS 的核心基础。

基础选择器

选择器类型 语法符号 示例代码 说明
标签选择器 标签名 p { color: blue; } 选中页面中所有的 <p> 标签。
类选择器 .类名 .nav { width: 100px; } 选中 class="nav" 的元素，最常用。
ID 选择器 #ID名 #header { height: 60px; } 选中 id="header" 的元素，具有唯一性。
通配符选择器 * * { margin: 0; padding: 0; } 选中页面中所有元素，常用于初始化。

CSS 引入方式

学会了如何“选中”元素，接下来我们要学习如何将 CSS 代码“放入”HTML 页面中。

三种引入方式对比

方式 语法位置 代码示例 特点
行内样式 标签的 style 属性 <h1 style="color:red;">标题</h1> 优先级最高，但结构与样式未分离，不推荐使用。
内部样式 HTML <head> 中的 <style> 标签 <style>h1 { color: red; }</style> 仅对当前页面有效，适合测试或单页专用样式。
外部样式 引入独立的 .css 文件 <link rel="stylesheet" href="style.css"> 推荐。结构与样式完全分离，可复用，易于维护。

避坑指南：

  • 使用外部样式表时，<link> 标签必须包含 rel="stylesheet" 属性，否则浏览器无法识别这是样式表，导致样式加载失败。

小贴士：实际开发中，99% 的情况都使用外部样式表。

选择器优先级（权重）

当一个元素被多个选择器（或多种引入方式）选中时，浏览器会根据优先级决定使用哪个样式。

权重计算规则（分值法）

我们可以给不同的选择器赋予不同的“分值”，总分越高，优先级越高：

  • ID 选择器：100 分
  • 类选择器：10 分
  • 标签选择器：1 分

举例计算：

  • #header .nav = 100 + 10 = 110 分 (胜出)
  • div .nav = 1 + 10 = 11 分

简单记忆口诀：行内 > ID > 类 > 标签

引入方式与优先级的关系

  • 就近原则：在引入方式中，行内样式优先级最高。
  • 后来者居上：如果是同一种引入方式（如都是内部样式），且选择器权重相同，后写的样式会覆盖先写的样式。

CSS 常用属性概览

文本属性

属性 说明 示例值
color 文本颜色 red, #333, rgb(0,0,0)
font-size 字号大小 16px, 1.5rem
font-weight 字体粗细 normal, bold, 700
text-align 文本对齐 left, center, right
text-decoration 文本装饰 none (去下划线), underline

背景属性

属性 说明 示例值
background-color 背景颜色 #f0f0f0
background-image 背景图片 url(images/bg.jpg)
background-repeat 背景平铺 no-repeat (不平铺)

答案：

  1. <link> 缺少 rel="stylesheet"。
  2. color 后面用了中文冒号 ：，且 red 后面漏了分号 ;。
  3. HTML 中是 id="header"，CSS 中应该用 #header，而不是 .header。

开发易错点总结

  1. 符号错误：CSS 中使用英文冒号 : 和分号 ;，漏写分号会导致后续样式失效。
  2. 选择器拼写：类名和 ID 名区分大小写，拼写必须与 HTML 中完全一致。
  3. 路径问题：引入外部 CSS 文件时，href 路径写错会导致样式加载失败（404）。
  4. 滥用 ID：ID 权重太高，不利于后期维护，建议尽量使用类选择器。
  5. 单位遗漏：除 0 以外，大多数数值属性（如 width, font-size）必须带单位（如 px）。

总结

  1. CSS 核心：负责网页美化，实现结构与表现分离。
  2. 语法基础：选择器 { 属性: 值; }。
  3. 选择器：熟练掌握 .类名 和 #ID名 的区别与使用场景。
  4. 引入方式：首选外部样式表 (<link>)，注意 rel 属性。
  5. 优先级：理解 行内 > ID > 类 > 标签 的层叠规则。
