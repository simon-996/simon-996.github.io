数据可视化第一课

发表于2026-03-01|更新于2026-03-04|课件
|浏览量:

课程须知

本课程是考查课程

课程安排：32学时| 2学分|16周

这门课学了有什么用

科研

不管是毕业论文、课程报告，还是科创项目，一份清晰的可视化图表能让你的成果 “一目了然”，比纯文字 / 表格更有说服力，分数和评审印象分都会更高

求职

数据分析、运营、产品、金融、互联网等岗位，招聘要求里几乎都有 “熟练使用 Python 做数据可视化”，这门课学完的实操能力，能直接写进简历，面试时拿出自己做的交互式图表作品集，远比空说 “我会 Python” 更有说服力。

生活

自己整理 “月度消费数据”，用 Matplotlib 画饼图看消费结构，用折线图看消费趋势，帮自己理性消费。

学什么

本学期是用 Python 做可视化，所以需要会 Python，以及相应的数据处理包 Numpy、Pandas

其实就是把文字的数据，如 excel，mysql表等数据可视化成图表。可以更直观的分析数据。

所以本学期主要学习的两大要点

  • 数据格式处理
  • 数据可视化

环境搭建

Python环境

运行 Python 代码。下载后直接安装就行，本学期我们使用 Anaconda

Anaconda 不是单一软件，而是Python 数据科学的一站式环境管理工具，可以理解为 “Python + 数据科学库 + 环境管理器” 的全家桶。

一键安装

不用手动装 Python 解释器，也不用挨个 pip 装 Matplotlib、Pyecharts、NumPy、Pandas 这些数据可视化必需的库 ——Anaconda 默认打包了近 200 个数据科学常用库，一键安装就能用

环境隔离

如果后续做不同项目（比如这门课用 Python 3.9，其他课需要 Python 3.7），可以用 Anaconda 创建 “独立环境”，避免库版本冲突（比如 A 项目需要 Pyecharts 1.9，B 项目需要 2.0，分开装不打架）；

Jupyter

Jupyter 是交互式编程笔记本，核心特点是 “代码 + 文字 + 图表” 一体化展示。

边写边跑，即时反馈

代码不用整段写完再运行，可按单元格分段执行（比如先写数据清洗代码，运行看结果，再写画图代码，立刻看到图表）。

vscode

vscode是轻量级但功能强大的通用代码编辑器，可以理解为 “升级版的记事本 + 代码助手”，支持 Python、Java 等几乎所有编程语言。

需要安装以下插件

  • Python Extension Pack
  • Jupyter
